How It Works

  1. Open a bank journal and click Import Statement.
  2. Upload a CSV or XLSX file.
  3. Review the detected columns and map them to Odoo fields.
  4. Adjust file-format and value-format options if needed.
  5. Run Test Import or import the statement directly.

Supported Capabilities

  • CSV import with configurable encoding, separator, and quote character
  • XLSX import with sheet selection
  • Date parsing for multiple common date formats
  • Decimal and thousands separator handling
  • Optional partner creation during import
  • Error policy: stop import or skip invalid rows
